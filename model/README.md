  # Physical CAD Design for EchoLog 
<p align="center">
   
The following print settings are recommended and universal for all printers. All parts can be printed at once. This table uses the filament provided in the BOM (**Overture PETG**); other brands may vary in temperature. Warping may be eliminated by using a draft shield.

### Basic Printing Config

| Property | Ideal | Minimum |
| :--- | :--- | :--- |
| **Layer Height** | 0.15 – 0.2mm | 0.25 – 0.3mm |
| **Infill** | 35%, Gyroid | 10%, Gyroid |
| **Perimeter** | 5 | 3 |
| **Support** | Organic, build plate only | Snug, build plate only |
| **Speed** | 200 – 600mm/s | 25 – 100mm/s |
| **Temperature** | 240°C, 80°C bed | 230°C, 70°C bed |

---

The following advanced print settings are also recommended but limited to the **Overture PET-G** [filament](https://a.co/d/02w19UyD) for reference. These properties were found and adjusted using the demo unit printed the week prior. Analysis of the demo unit showed signs of **under-extrusion** at some points, which will severely compromise the IP rating. This has been taken into account in the following table and is universal to common modern printers.

### Advanced Printing Config

| Property | Value |
| :--- | :--- |
| **'Elephant's foot' Compensation** | 0.15mm |
| **Maximum Volumetric Speed** | 8mm^3/s |
| **Wall Generator** | Classic |
| **Infill overlap (Anchor)** | >30% |
